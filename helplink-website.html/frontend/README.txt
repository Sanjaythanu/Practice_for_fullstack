# SST E-Commerce Website

## Backend Setup
1. Open a terminal and navigate to the `backend` folder.
2. Run `npm install` to install dependencies.
3. Make sure MongoDB is running locally (or update the `MONGO_URI` in `.env`).
4. Run `npm start` (or `npm run dev` for auto-reload with nodemon).

## Frontend Setup
1. Open `frontend/index.html` in your browser.
2. The frontend will connect to the backend at `http://localhost:5000`.

## Features
- Browse products
- Add to cart
- Register/Login
- View cart

You can extend this project with product management, order checkout, admin dashboard, etc.
